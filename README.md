# keldon-blog

Source content for the keldon blog, published at [keldon.io/blog](https://keldon.io/blog/).

## What's in here

This repository contains the Markdown source for blog posts — release announcements, technical deep dives, and project updates.

## Writing a post

Posts are Markdown files with front matter:

```yaml
---
title: "Your post title"
date: 2026-06-30
author: "Your Name"
authorURL: "https://www.linkedin.com/in/elhimad"
summary: "A one-sentence summary shown on the blog index."
---
```

## Contributing

Guest posts and technical write-ups about running Keldon in the real world are welcome.
